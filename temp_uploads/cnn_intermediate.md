Convolutional Neural Networks (CNN)

1. Introduction:
Convolutional Neural Networks are a type of deep learning model mainly used for image processing and computer vision tasks.

2. Key Idea:
CNNs automatically detect important features from images such as edges, shapes, and textures.

3. Architecture:
- Convolution Layer: Applies filters to extract features
- Activation Function: Usually ReLU
- Pooling Layer: Reduces image size (Max Pooling)
- Fully Connected Layer: Produces final output

4. How it Works:
CNN scans an image using filters and builds feature maps that help in identifying patterns.

5. Important Concepts:
- Filters/Kernels
- Stride
- Padding
- Feature Maps

6. Applications:
- Image classification
- Object detection
- Face recognition
- Medical image analysis

7. Advantages:
- Automatically extracts features
- High accuracy in image tasks

8. Limitations:
- Requires large datasets
- Computationally expensive
