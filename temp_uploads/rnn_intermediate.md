Recurrent Neural Networks (RNN)

1. Introduction:
Recurrent Neural Networks are a type of neural network designed for sequential data.

2. Key Idea:
RNNs have memory, meaning they can use previous information to influence current output.

3. Architecture:
- Input Layer
- Hidden Layer (with feedback loop)
- Output Layer

4. How it Works:
RNN processes one element at a time while maintaining a hidden state that carries information from previous steps.

5. Important Concepts:
- Hidden State
- Sequence Data
- Time Steps
- Backpropagation Through Time (BPTT)

6. Variants:
- LSTM (Long Short-Term Memory)
- GRU (Gated Recurrent Unit)

7. Applications:
- Text generation
- Language translation
- Speech recognition
- Time series prediction

8. Advantages:
- Good for sequential data
- Remembers previous inputs

9. Limitations:
- Vanishing gradient problem
- Hard to train for long sequences
